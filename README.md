# Bookshelf Project (Flask)

## Run steps

1. Open project in VS Code
2. Create virtual env (optional):
   python -m venv venv
   venv\Scripts\activate   (Windows)

3. Install requirements:
   pip install -r requirements.txt

4. Run:
   python app.py

5. Open:
   http://127.0.0.1:5000

## Features
- Signup/Login
- Sell used books (with image upload)
- Marketplace listing
- Cart (no payment gateway)
- Notes sharing
- Profile page
